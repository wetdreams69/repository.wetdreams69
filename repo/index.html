Repositorio Kodi - wetdreams69

📥 Instalación del Repositorio

Para empezar, necesitas instalar el Repositorio en tu Kodi. Podés descargar el archivo ZIP a continuación, o agregar esta página como fuente.

Opción 1: Descargar el archivo ZIP e instalarlo manualmente en Kodi:

📦 Descargar repository.wetdreams69-1.0.0.zip

Opción 2: Agregá esta URL como Fuente en el Gestor de Archivos de Kodi:

https://wetdreams69.github.io/repository.wetdreams69/

Addons disponibles en el repositorio:

(Una vez instalado el repositorio arriba, busca e instala estos addons directamente desde Kodi > Instalar desde Repositorio > wetdreams69 Repository).

  • Plugin MercadoPlay (plugin.video.mercadoplay)
    Addon para ver el catálogo y contenido de MercadoPlay.
  • Plugin YouTube Latino (plugin.video.youtubemovieslatino)
    Películas latinas completas alojadas en YouTube.
  • Plugin Pluto VOD (plugin.video.pluto_vod)
    Addon para ver contenido y catálogo On Demand de Pluto TV.
