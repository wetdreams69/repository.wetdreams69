# Pluto VOD - Kodi Add-on

A powerful Kodi add-on for watching Pluto TV content with automatic ad-skipping.

## Features

✨ **Search & Browse** - Find movies and TV series easily  
📺 **Watch Anywhere** - Stream to your Kodi media center  
🚫 **Auto Ad-Skip** - Automatically skip short ads (15-45 seconds)  
🔒 **DRM Protected** - Secure streaming with Widevine support  
⚡ **Fast & Responsive** - Quick loading and navigation  

## Installation

1. Download the add-on repository
2. Install via Kodi: Add-ons → Install from file
3. Select `plugin.video.pluto_vod`
4. Enable the add-on and enjoy!

## How to Use

### Search for Content
1. Open the add-on from the Video menu
2. Click "Search" 
3. Enter a movie or series name
4. Browse results

### Watch a Series
1. Select a series from search results
2. Choose a season
3. Pick an episode
4. Hit play!

### Watch a Movie
1. Search for a movie
2. Click to play immediately

## Features Explained

### 🚫 Auto Ad-Skip
The plugin automatically detects and skips short ads that appear during playback:
- Detects ads between 15-45 seconds
- Skips them automatically
- Shows a notification for each skipped ad
- Works seamlessly in the background

### 🔒 Secure Streaming
- Uses industry-standard Widevine DRM encryption
- Authenticates with Pluto TV servers
- Maintains secure session tokens
- Protects against unauthorized access

## Requirements

- Kodi 18.0 or higher
- `inputstream.adaptive` add-on (optional but recommended)
- Internet connection
- Supported location (Pluto TV availability)

## Troubleshooting

### Nothing appears when I search
- Check your internet connection
- Try a simpler search term
- Make sure you're in a supported region

### Playback won't start
- Verify `inputstream.adaptive` is installed
- Check that `inputstream.adaptive` is enabled
- Restart Kodi

### Ads aren't being skipped
- Some ads may not be detectable
- Auto-skip works best on live streams
- You can manually skip using your remote
