# Angulismo TV

Angulismo TV is a premium Kodi addon designed to provide a smooth and reliable streaming experience. It features an exclusive "Health System" that indicates exactly which signals are working in real-time, so you don't waste time clicking on dead links.

## Key Features

- **Smart Indicators**: Know if a channel is working before opening it thanks to our color-coding system.
- **Ultra-Fast Loading**: Optimized for instant channel list and source retrieval.
- **Automatic Updates**: The addon searches for the best servers and logos for you, with no configuration required.
- **Maximum Compatibility**: Works perfectly on Kodi 19, 20, and the new Kodi 21 (Omega).

## What do the colors mean?

When viewing channel options, you will notice different colors. This is our "Health System":

- **🟢 Green**: The signal is working perfectly and has been recently verified.
- **🟡 Yellow**: The signal might be unstable or only works intermittently.
- **🔴 Red**: The signal is currently down or has consistent failures.
- **⚪ White**: This is a new source that hasn't been tested by the system yet.

## Requirements

To enjoy Angulismo TV, you need:
- Kodi 19 or higher.
- A stable internet connection.
- `InputStream Adaptive` enabled in Kodi settings (usually enabled by default).

## Installation

1. Download the `plugin.video.angulismo.zip` file.
2. Open Kodi and go to the **Add-ons** section.
3. Click on the **Box Icon** (Add-on browser) at the top left.
4. Select **Install from zip file**.
5. Locate the downloaded file and click OK.
6. Wait for the "Add-on installed" notification to appear.

## Maintenance

If you notice the channel list is not updating or the colors seem "frozen," you can use the **Clear Cache** option in the addon settings to refresh all information.
